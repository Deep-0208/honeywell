## How to Read This Map

| Column | Meaning |
|---|---|
| **Keyword** | The exact search query to target |
| **Intent** | **T** = Transactional (ready to buy/inquire) · **CI** = Commercial Investigation (researching options) · **I** = Informational (learning) · **L** = Local (location-specific) |
| **Difficulty** | 1–10 scale. **1–3** = Low (rank in 1–3 months) · **4–6** = Medium (3–6 months) · **7–10** = High (6–12+ months) |
| **Biz Value** | 1–10 scale. **8–10** = Directly generates quote requests · **5–7** = Builds authority/trust that leads to sales · **1–4** = Supports SEO but indirect conversion |
| **Page** | The specific URL this keyword is assigned to (from the website architecture) |

> [!IMPORTANT]
> **Rule: One primary keyword per page.** Each page targets ONE primary keyword and 2–4 secondary keywords. No two pages should compete for the same primary keyword (keyword cannibalization). Secondary keywords appear naturally in H2s, body text, and FAQ sections.

| **No cannibalization** | No two pages target the same primary keyword. Use the assignment table above as the single source of truth. |
