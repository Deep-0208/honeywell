- **H1** — Single `<h1>` per page (unique, keyword-targeted)
- **Primary Keyword** — The #1 keyword this page is built to rank for
- **Search Intent** — T (Transactional) · CI (Commercial Investigation) · I (Informational) · N (Navigational) · L (Local)
- **Priority** — 🔴 Critical (launch blocker) · 🟠 High (Week 1-2) · 🟡 Medium (Week 3-4) · 🟢 Low (Month 2+)
