# Phase-by-Phase Roadmap (Updated)

This roadmap has been updated to reflect the removal of all Services and Applications pages from the website architecture.

## Phase 1: Core Foundation & UI Component Library (Weeks 1-2)
- **Goal:** Set up Next.js App Router, Tailwind CSS, global layouts, headers, footers, and typography.
- **Tasks:**
  - Build navigation without Services/Applications dropdowns.
  - Implement the UI primitives (Buttons, Cards, Forms).
  - Setup MDX parsing logic for Products and Industries.

## Phase 2: Product Silos (Weeks 3-4)
- **Goal:** Build the primary revenue-driving pages.
- **Tasks:**
  - `Hydraulic Cylinders` (Pillar + 8 products)
  - `Hydraulic Power Packs` (Pillar + 7 products)
  - `Manifold Blocks` (Pillar + 4 products)
  - `Hydraulic Systems` (Single page)
  - Populate with real technical specs, imagery, and SEO metadata.

## Phase 3: Industry & Location Pages (Weeks 5-6)
- **Goal:** Capture targeted, high-conversion long-tail traffic.
- **Tasks:**
  - Develop the 10 Industry pages (e.g., Injection Moulding, Automotive, Steel).
  - Develop the 12 Location pages (Ahmedabad, Surat, India-wide).
  - Cross-link Industries to Products.

## Phase 4: Content Hub & Resources (Weeks 7-8)
- **Goal:** Build topical authority and E-E-A-T.
- **Tasks:**
  - Publish 25 Blog articles.
  - Publish 5 Case Studies.
  - Setup FAQ, Downloads, Gallery, and Calculators.

## Phase 5: Launch & Technical SEO QA (Week 9)
- **Goal:** Final polish and deployment.
- **Tasks:**
  - Verify JSON-LD Schema on all 95+ pages.
  - Test contact and quote forms.
  - Submit sitemap to Google Search Console.
