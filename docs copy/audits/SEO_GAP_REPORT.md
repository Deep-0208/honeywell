## CRITICAL SEO GAPS BY CATEGORY

### 🔴 GAP 1 — Missing Product OG Images (4 pages)

**Affected Pages:**
- `/products/hydraulic-cylinders/` → references `/images/products/hydraulic-cylinders-ahmedabad.webp` — **FILE DOES NOT EXIST**
- `/products/hydraulic-power-packs/` → references `/images/products/hydraulic-power-pack-manufacturer.webp` — **FILE DOES NOT EXIST**  
- `/products/manifold-blocks/` → references `/images/products/custom-hydraulic-manifold-block.webp` — **FILE DOES NOT EXIST**
- `/products/hydraulic-systems/` → references `/images/products/custom-hydraulic-systems-manufacturer.webp` — **FILE DOES NOT EXIST**

**Impact:** Google cannot generate Open Graph cards for these pages. Social sharing will show broken/default images. OG image schema will reference broken URLs. **Priority: CRITICAL**

---

### 🔴 GAP 2 — Schema: Product Hub Pages Missing Organization Reference

**Issue:** Product hub pages only have: `WebPage`, `BreadcrumbList`, `FAQPage`, `CollectionPage`  
**Missing:** No `@id` reference to `/#organization` or `/#localbusiness` schemas defined on homepage  
**Fix Required:** Add `"publisher": {"@id": "https://www.honeywellhydraulics.com/#organization"}` to each WebPage schema's `about` or `publisher` node

---

### 🔴 GAP 3 — CollectionPage Schema References Unbuilt Pages

**Issue:** All 4 hub pages have `CollectionPage.hasPart[]` arrays pointing to individual product pages that do NOT yet exist.

**Examples:**
- `/products/hydraulic-cylinders/double-acting/` → **404**
- `/products/hydraulic-cylinders/single-acting/` → **404**
- `/products/hydraulic-power-packs/3-phase/` → **404**
- `/products/manifold-blocks/cetop/` → **404**
- 17+ more broken schema references

**Impact:** Google follows these schema URLs and finds 404s — signals poor site quality. Remove or comment out `hasPart` until pages are built.

---

### 🟠 GAP 4 — No /products/ Hub Index Page

**Issue:** The URL `/products/` does not have a built page. The sitemap references it as a breadcrumb step. Clicking "Products" in breadcrumb schema will return a 404.  
**Priority:** High — build this before Phase 5

---

### 🟠 GAP 5 — URL Structure Mismatch (Sitemap vs Built)

**Sitemap Config** references `/hydraulic-cylinders/` (no `/products/` prefix)  
**Built URLs** use `/products/hydraulic-cylinders/` (with prefix)  
**Impact:** Sitemap priority overrides will never match built page paths — all hub pages get default priority instead of 0.9

**Fix:**
```js
// In next-sitemap.config.js, line 57 — update product pillars array:
const productPillars = [
  '/products/hydraulic-cylinders',
  '/products/hydraulic-power-packs', 
  '/products/manifold-blocks',
  '/products/hydraulic-systems'
];
```

---
